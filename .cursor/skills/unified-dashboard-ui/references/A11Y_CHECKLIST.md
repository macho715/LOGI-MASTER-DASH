## WCAG 2.2 AA 체크리스트

### 키보드 접근성
- [ ] 모든 인터랙티브 요소 키보드 접근 가능
- [ ] ESC로 드로어/모달 닫기
- [ ] 포커스 순서 논리적

### 대비
- [ ] 텍스트 대비 ≥4.5:1
- [ ] 상태 색상 + 텍스트/아이콘으로 구분

### ARIA
- [ ] aria-live 영역 (KPI 업데이트)
- [ ] aria-label/aria-labelledby 적절히 사용
- [ ] 역할(role) 명시

### 모바일
- [ ] 터치 타겟 ≥44x44px
- [ ] 제스처 대체 방법 제공

### 테스트
- [ ] 스크린 리더 테스트
- [ ] 키보드만으로 전체 플로우 테스트
